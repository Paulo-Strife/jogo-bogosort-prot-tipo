# jogo-bogosort-prot-tipo
Este é um diretório de estudos destinado á um jogo que eu criei baseado no algoritmo de ordenação bogosort

# Como o Jogo Funciona?
O jogo é um jogo de apostas, então você começa com 100 créitos, seu objetivo é chegar em 200 créditos, vence o jogador que conseguir chegar em 200. Entretando, você deve apostar o valor correto de ordenações que o bogosort vai levar para odernar o array, do contrário você perderá sua rodada.
